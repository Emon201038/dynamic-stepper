# Dynamic-stepper

a simple dynamic stepper template for use various purpose.

## Demo live link here:

https://emon-dynamic-stepper.vercel.app/

## Run Locally

Clone the project

```bash
  git clone https://github.com/Emon201038/dynamic-stepper.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

or

```bash
  npm run dev
```
